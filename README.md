📂 Автоматичний зміст

Головна сторінка проекту (index.html) використовує GitHub API для динамічного створення списку всіх файлів та папок у цьому репозиторії. Вам не потрібно оновлювати зміст вручну — він оновлюється автоматично при додаванні нових матеріалів.
